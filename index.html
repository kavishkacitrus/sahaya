Why Choose Sahaya?

Designed for reliability in any environment, Sahaya ensures you can communicate when it matters most, without dependency on external networks.

True Offline Chat

Send and receive messages directly between devices. Perfect for remote areas, crowded events, or emergency situations.

Zero Configuration

The app automatically detects and connects with other Sahaya users in range using robust Wi-Fi Direct technology.

Privacy-First Security

Messages are exchanged locally, peer-to-peer. No servers, no data logging, ensuring maximum user privacy and control.

How to Get Started

Follow these simple steps to download, install, and start using Sahaya for offline communication

Download & Installation

  • Click the download button above to get the .apk file
  • Open the downloaded .apk file to begin installation
  • If prompted, allow installation from unknown sources in your device settings
  • Follow the on-screen instructions to complete installation
  • Grant necessary permissions when the app requests them

How to Use Sahaya

  • Start the app and choose to act as a Server or Client
  • Discover nearby devices running Sahaya
  • Tap a device from the list to connect
  • Start chatting and sharing data instantly
  • All messages work offline using Wi-Fi Direct technology

Our Development Roadmap

We are continuously working to enhance Sahaya's capabilities and bring critical new features to help users connect when it matters most.

Phase 1: Direct P2P Communication

Enabled nearby devices to discover and communicate via Wi-Fi Direct without reliance on any external network infrastructure.

Status: Achieved

Phase 2: Emergency Mesh Network (Sahaya SOS)

Develop a message chain system for emergencies. If a user is without internet, a distress signal (SOS) can be relayed from device to device until a connection is reached.

The connected device will then securely update the Sahaya website with the SOS message and location for immediate help.

Status: In Development (Coming Soon)

Phase 3: Release on Google Play Store

Prepare and publish Sahaya officially on the Google Play Store, ensuring full compliance with Android policies, security requirements, and performance benchmarks.

Includes app signing, privacy disclosures, content guidelines, and preparing promotional material for the store listing.

Status: Planned
